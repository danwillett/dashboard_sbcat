# 🗺️ Active SB Dashboard (Vite + React + ArcGIS)

This is the Vite-based version of the Active SB Dashboard: a fast, modern frontend for exploring Santa Barbara’s biking and walking data. It uses React, ArcGIS JS API, and Calcite Web Components — all bundled through Vite for blazing dev speed.

---

## ⚡ Tech Stack

- **Vite** – Modern dev server and bundler
- **React 18** – Component framework
- **TypeScript** – Static typing
- **React Router** – Client-side routing
- **@arcgis/core** – ArcGIS Maps & widgets
- **@arcgis/map-components** – Web components for maps
- **@esri/calcite-components** – UI components from Esri
- **MUI (Material UI)** – Custom theming
